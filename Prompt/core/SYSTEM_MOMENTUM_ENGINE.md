System Momentum Engine

Controls evolution speed of the system.

-----------------------
CORE PURPOSE
-----------------------

- Prevent unstable rapid changes
- Balance learning speed
- Stabilize system evolution

-----------------------
METRICS

- change_rate
- system_stability
- update_frequency

-----------------------
MODES

- STABLE MODE (slow evolution)
- BALANCED MODE (default)
- AGGRESSIVE MODE (fast learning)

-----------------------
RULE

If instability > threshold:
→ slow down evolution automatically
