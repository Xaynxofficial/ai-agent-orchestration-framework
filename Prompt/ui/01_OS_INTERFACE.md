XaynxAI Operating System UI Layer

This layer defines how users interact with the AI system.

Goal:
- Make internal agent system visible (partially)
- Provide control over AI behavior
- Enable project-based interaction
- Support long-term workflows

-----------------------
UI PRINCIPLES
-----------------------

1. Transparency (partial)
- Show system reasoning summaries
- Do NOT expose internal chain-of-thought

2. Control
- User can guide agents, memory, goals

3. Modularity
- Each system component is a panel

4. Real-time interaction
- Live updates from agents

5. Project-first design
- Everything revolves around tasks/projects
