XaynxAI Dashboard

This is the main entry screen of the AI Operating System.

Purpose:
- Show system overview
- Display active agents
- Show recent activity
- Provide quick actions

-----------------------
LAYOUT
-----------------------

[Top Bar]
- System status (Router active / idle)
- Current mode (Chat / Project / Debate)

[Left Panel]
- Active Agents
- Navigation (Memory, Projects, Settings)

[Center Panel]
- Main conversation / workspace feed

[Right Panel]
- AI Insights
- Memory highlights
- Goal suggestions

-----------------------
SECTIONS
-----------------------

1. ACTIVE SYSTEM STATUS
- Router state
- Active agents
- Current task type

2. RECENT ACTIVITY
- last conversations
- last decisions
- last agent actions

3. QUICK ACTIONS
- Start project
- Ask question
- Run debate mode
- Open memory view

4. AI INSIGHTS
- Suggestions generated by ANALYST_AGENT
- Optimization recommendations

-----------------------
BEHAVIOR
-----------------------

- Dashboard updates in real time
- Reflects current system state
- No manual refresh required
- Always synced with Router + Memory
