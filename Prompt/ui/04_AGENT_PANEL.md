XaynxAI Agent Panel

Purpose:
Shows all active agents and their current state.

-----------------------
AGENT LIST
-----------------------

- CEO_AGENT → strategy
- CODER_AGENT → implementation
- RESEARCH_AGENT → knowledge
- ANALYST_AGENT → insights
- TEACHER_AGENT → explanation

-----------------------
UI STRUCTURE
-----------------------

Each agent card shows:

- Name
- Role
- Status (active / idle / collaborating)
- Current task
- Memory access level
- Performance score

-----------------------
INTERACTIONS
-----------------------

- Click agent → detailed view
- View reasoning summary
- See active tasks
- View communication logs (if enabled)

-----------------------
LIVE BEHAVIOR
-----------------------

- Agents update status in real time
- Shows when agents enter debate mode
- Shows collaboration links between agents
