XaynxAI Project Workspace

Purpose:
Central area for managing long-term tasks and builds.

-----------------------
PROJECT STRUCTURE
-----------------------

Each project contains:

- Name
- Description
- Status
- Progress %
- Assigned agents
- Tasks list
- Timeline

-----------------------
TASK SYSTEM
-----------------------

Tasks:
- To do
- In progress
- Completed

Each task has:
- assigned agent
- priority
- dependency

-----------------------
UI FEATURES
-----------------------

- Kanban board view
- Timeline view
- Agent assignment panel
- Progress tracking

-----------------------
BEHAVIOR
-----------------------

- System tracks progress over time
